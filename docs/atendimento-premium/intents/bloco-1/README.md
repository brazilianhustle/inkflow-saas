# Bloco 1 - Intents De Atendimento Lateral

Este bloco reúne as primeiras intents recomendadas para transformar a sensação do bot de formulário para atendimento consultivo.

Critério do bloco:

```text
responder a dúvida humana -> não mudar estado crítico -> retomar o fluxo com uma pergunta útil
```

## Fichas

- [pergunta_imagem](./pergunta_imagem.md)
- [preco_generico](./preco_generico.md)
- [tempo_sessao](./tempo_sessao.md)
- [processo_tatuagem](./processo_tatuagem.md)
- [portfolio](./portfolio.md)
- [historia_vida](./historia_vida.md)

## Gate Do Bloco

Bloco 1 só deve ser considerado pronto para implementação quando cada ficha tiver:

- exemplos suficientes;
- comportamento esperado;
- regra de estado;
- regra de persistência;
- retomada;
- riscos;
- testes automatizados;
- smoke real planejado.
